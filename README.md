# FluctSDK

## Installation

FluctSDK は [CocoaPods](http://cocoapods.org) から利用可能です

利用するプロジェクトの`Podfile`に下記の記述を追加して下さい

```ruby
use_frameworks!

pod "FluctSDK"
```

## APIドキュメント
FluctSDK-iOSの[Wiki](https://github.com/voyagegroup/FluctSDK-iOS/wiki)を参照してください
